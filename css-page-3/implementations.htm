CSS3 Paged Media Module Implementation Report

This implementation report will document the interoperability of multiple implementations of the features of the CSS3 Paged Media Module Candidate Recommendation. It will be created by the CSS Working Group (Members only) during the Candidate Recommendation phase. The Working Group expects to release the test suite for CSS3 Paged Media Module within three months from the publication of the CSS3 Paged Media Candidate Recommendation.

In order to exit the Candidate Recommendation phase, the following criteria must be satisfied:

  1. At least two implementations of all the required features of this specification.
  2. A minimum of six months of the CR period must be elapsed to ensure that enough time is given for providing implementation feedback.

Appendix C. Changes

This appendix is informative, not normative.

CSS 2.2 is an updated revision of CSS2. The changes between the CSS 2.1 specification (see [CSS2]) and this specification fall into five groups: known errors, typographical errors, clarifications, changes and additions. Typographical errors are not listed here.

This chapter is not a complete list of changes. Minor editorial changes are not listed here.
